# LifeOS PRD

## Vision
LifeOS is a calm, intelligent operating system for modern life. It helps people balance time, money, and energy by combining calendar planning, task execution, habit consistency, budgeting, and goal tracking in one place.

## Problem
People often manage their lives across disconnected tools: calendar, task manager, budget app, habit tracker, and notes. That fragmentation makes it hard to see how time usage and spending behavior influence each other.

## Goals
- Help users make better decisions with a unified daily view.
- Connect productive time with financial habits.
- Reduce cognitive load through simple automation and intelligent suggestions.
- Encourage consistency by making progress visible and rewarding.

## Core user journeys
1. Start the day with a focused plan.
2. Track tasks, habits, and spending from a single dashboard.
3. Receive gentle nudges about upcoming bills, deadlines, and momentum.
4. Review progress through weekly or monthly insights.

## MVP scope
- Dashboard with daily summary cards.
- Task and habit overview.
- Budget snapshot and spending trend.
- Smart insight cards.
- Basic authentication framing and polished UI shell.

## Non-goals for MVP
- Full social network.
- Advanced AI coaching.
- Real-time collaboration.
- Complex calendar drag-and-drop.

## Success metrics
- Daily active users in the first release.
- Users complete at least one habit or task per day.
- Users revisit the dashboard multiple times a week.
